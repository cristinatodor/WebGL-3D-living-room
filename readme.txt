The sent folder contains 3 sub-folders. 
The source codes are in a folder called src. 

How to run the system:
Open a terminal window and type in the following command:

(MacOs)
/Applications/Google\ Chrome.app/Contts/MacOS/Google\ Chrome --allow-file-access-from-files

(Windows)
C:\Program Files (x86)\Google\Chrome\Application\chrome.exe" --allow-file-access-from-files

This will open a Chrome window.
Then, simply open the LivingRoom.html from the src folder into this window. The textures will load when pressing one of the keys specified in the keyboards controls list found on the right part of the page. Use these controls for interaction with the living room.
